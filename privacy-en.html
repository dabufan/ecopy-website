Privacy Policy

Data Collection and Use

Important Notice: ecopy is a completely local application. All data is stored on your device and never uploaded to any server.

Types of Data We Collect

ecopy only collects the following types of data:

  • Other User Content: Text, images, and file content you copy
  • App Settings: Your preferences, shortcut configurations, etc.
  • Usage Statistics: App usage data (only for improving user experience)

Data Usage

We collect data only for the following purposes:

  • Providing clipboard management functionality
  • Saving your app settings and preferences
  • Improving app performance and user experience
  • Providing technical support services

Data Storage

All data is stored locally on your device:

  • Text records: Stored in local database
  • Image files: Saved in app sandbox directory
  • Settings information: Stored in system preferences
  • Never uploaded to any external server

Data Security

We take the following measures to protect your data security:

  • All data is stored only on your device
  • Using system-provided secure storage mechanisms
  • No personal identification information is collected
  • No data sharing with other apps or services

Third-Party Services

ecopy does not integrate any third-party services, including:

  • No advertising services
  • No analytics services
  • No cloud sync services
  • No social sharing features

Data Deletion

You can delete app data at any time:

  • Delete individual records within the app
  • Clear all data in settings
  • Uninstalling the app will delete all data
  • Data cannot be recovered after deletion

Children's Privacy

ecopy is suitable for users of all ages:

  • No collection of children's personal information
  • All data is stored only on local devices
  • Parents have complete control over data access

Privacy Policy Updates

We may update this privacy policy from time to time:

  • Major changes will be notified through app updates
  • We recommend checking the latest version regularly
  • Continued use of the app indicates acceptance of the updated policy

Contact Us

If you have any questions about this privacy policy, please contact us through:

  • GitHub: https://github.com/dabufan/ecopy
  • We will respond to your inquiries within 24 hours
